Para el backend, instalar:
    npm install express bcrypt cors dotenv jsonwebtoken mime-types multer mysql2 sharp socket.io helmet express-rate-limit zod openai

Todos los fontends con vue deben ser creados con: dependencias de producción y desarrollo:
    npm install @nuxt/ui vue-i18n socket.io-client @capacitor/core @capacitor/app
    npm install -D tailwindcss @capacitor/cli stylus
    npm install mammoth xlsx three
Crear:
    npm create vue@latest
